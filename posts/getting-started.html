# Quick Start

最快的方式直接用 create-vuepress-site generator，來直接 build 出官方給我們的一個模板！

之後會詢問我們一些對於 site 的 configure（ Project name, Description, Maintainer Email, Maintainer Name, Repository URL）

cd docs
yarn install & yarn dev

# Manual Installation

當然也可以選擇手動的方式來建立自己的 VuePress！

  1. 建立新的 directory
mkdir vuepress-starter && cd vuepress-starter
  1. init package manager，這裡使用 yarn
yarn init
  1. install VuePress
yarn install -D vuepress

如果要在本來的專案引進 VuePress 就直接來到step3就行了！

  1. 建立第一個doc
mkdir docs && echo '# Hello VuePress' > docs/README.md
  1. 到 package.json 加入 scripts
 "scripts": {
    "dev": "vuepress dev src",
    "build": "vuepress build src"
  },
  1. 打開來！！！！！
yarn dev

VuePress 會default 在8080，啟動hot-reloading dev server

就 4 這麼 EZ 😄
