The front matter must be the first thing in the file and must take the form of valid YAML set between triple-dashed lines. -- https://jekyllrb.com/docs/front-matter/

# permalink

A permalink is a URL that is intended to remain unchanged for a long time

Default: /:regular

date: 2022-01-05
permalink: /:year/:month/:day/:slug

設定完可以發現這頁的 url 變成了 http://localhost:8080/2022/01/05/markdown/

可以用 Global Computed 來抓取一些資訊！ ($site、$page、$frontmatter)

# Advanced Configuration

可以透過 config 去客製化 markdown-it (Markdown renderer)

module.exports = {
  markdown: {
    // options for markdown-it-anchor
    anchor: { permalink: false },
    // options for markdown-it-toc
    toc: { includeLevel: [1, 2] },
    extendMarkdown: md => {
      // use more markdown-it plugins!
      md.use(require('markdown-it-xxx'))
    }
  }
}
